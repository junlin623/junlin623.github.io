title: ‘IDEA中HTML注释顶格的问题-junlin623’
date: 2023-03-06 09:12:00

idea中在html文件中使用注释时如果不在行开头添加注释的话就会出现上图的情况，体验很不好

解决方法：

File -> Settings -> Editor -> Code Style -> HTML

将箭头所指的取消勾选即可

==在xml中进行以上设置，会发现没有用，这时候需要使用 Ctrl + Shift + / 才会有效，Ctrl + / 还是会顶格，不知道为啥==
