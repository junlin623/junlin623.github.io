title: ‘使用登录过的cookie信息来实现另一个连接中的直接登录-junlin623’
date: 2023-03-07 00:01:00

为了批量获取并编辑浏览器中对应网页的cookie信息，可以安装一个插件 Cookie-Editor

在Edge浏览器中直接搜索并添加即可

如果想要在 Chrome 中安装此插件，可以在谷歌应用商店或者第三方网站下载，也可以直接利用 Edge 中添加好的插件直接导入到 Chrome 中，

Edge 浏览器中插件的默认安装位置： C:\Users\用户名\AppData\Local\Microsoft\Edge\User Data\Default\Extensions

通过查看已安装插件的 ID 信息，找到对应插件：

进入该目录可以看到记录版本号的文件夹：

回到Edge浏览器中选择 ==打包扩展==

选择扩展目录：

“私钥文件”是可选项，不需要选择，直接点击“打包扩展”，点击确定，

浏览器扩展程序导出成功，会在对应插件根目录下创建了两个文件：

将得到的crx文件直接拖拽到页面中即可

但是会有错误，无法正常使用，

需要将 .crx 文件改为 .rar 文件，然后提取文件到一个文件夹， 可能会提示头部错误，不用理会

选择解压出来的文件夹即可
